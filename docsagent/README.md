| `04-DATABASE.md` | Schema database, model, dan relasi |
| `05-CONFIGURATION.md` | Semua konfigurasi environment variable |
| `06-DEVELOPMENT.md` | Panduan development, testing, coding convention |
| `07-DEPLOYMENT.md` | Panduan deployment production |
| `08-ERROR-HANDLING.md` | Sistem error handling dan kode error |
